# Privacy Policy

## Data Handling

NeuroStack is a **local-first** tool. Your vault data stays on your machine.

### Lite Mode
- All processing happens locally
- No network calls required
- Your data never leaves your machine

### Full Mode
- Embeddings and summaries are generated by **local** Ollama models
- No data is sent to cloud services by default
- Cloud API usage (if enabled) is opt-in with explicit warnings

### What We Don't Collect
- No telemetry
- No analytics
- No usage tracking
- No crash reports sent externally

### Your Data
- Vault content is indexed into a local SQLite database
- The database is stored at `~/.local/share/neurostack/neurostack.db`
- Delete the database at any time to remove all indexed data
- Uninstalling NeuroStack removes all traces
