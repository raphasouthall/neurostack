# Scoping Review vs Systematic Review

Scoping reviews and systematic reviews serve different purposes despite sharing surface similarities. Choosing the wrong one wastes effort and produces the wrong kind of answer.

## When to Use Each

| Criterion | Scoping Review | Systematic Review |
|-----------|---------------|-------------------|
| Question type | Broad, exploratory | Narrow, focused |
| Goal | Map the landscape | Answer a specific question |
| Quality appraisal | Optional | Required |
| Synthesis | Descriptive | Quantitative (meta-analysis) or structured narrative |
| Field maturity | Emerging, heterogeneous | Established, sufficient studies |
| Output | Gap map, typology | Effect estimate, recommendation |

## Decision Rule

Ask: "Do I know enough to write a precise, answerable question?"
- **Yes** → systematic review
- **No** → scoping review first, then refine into systematic review

## Frameworks

- **Scoping**: Arksey & O'Malley (2005), updated by Levac et al. (2010). Reporting: PRISMA-ScR
- **Systematic**: Cochrane Handbook. Reporting: PRISMA 2020
